# cricket_win_predictor
model which predict the winner of the match and how to win according to the situation.
This project involves the development of a high-fidelity Predictive Analytics Dashboard that calculates the win probability of a chasing team in T20 cricket. Unlike basic models, this engine incorporates match momentum, pitch conditions, and player impact factors to provide a 360-degree view of the match state. It serves as a Decision Support System (DSS) for analysts to simulate "What-If" scenarios in real-time.
